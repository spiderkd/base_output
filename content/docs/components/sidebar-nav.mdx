## Item definition

```tsx
interface NavItem {
  id: string
  label: string
  href?: string             // optional — used for <a> href
  icon?: ReactNode          // optional inline SVG
  badge?: string | number   // optional badge value (right-aligned)
}
```

## Design notes

Active item: a 4px solid primary left strip grows from `width: 0` via CSS transition, plus 10% primary tint background. Uses `position: absolute; left: 0` — not CSS `border-left`, which would shift the layout.

Hover: a 2px secondary left strip fades in with 12% secondary tint background.

Section headers sit above a double-rule at reduced opacity (secondary 40% top, primary 20% bottom) — quieter than the full Separator component.

Badge values use solid primary fill if active, solid secondary fill otherwise — 8px uppercase, flush right.
