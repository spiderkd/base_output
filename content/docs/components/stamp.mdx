## Design notes

The stamp is built entirely in SVG — two concentric `<circle>` elements with `stroke-dasharray` to create the serrated border. The outer circle uses the main ink color; the inner circle uses a complementary ink at lower opacity. Their `strokeDasharray` values are offset so the dashes don't align — the inner and outer teeth interlock like a real stamp die.

The `filter: url(#riso-bleed-soft)` makes the stroke slightly ink-bleed at its edges — a soft version of the same bleed filter used on avatar circles.

The `rotate` prop (default varies by variant) applies a small CSS rotation to the container, making the stamp look hand-placed. Rotation is applied *outside* the SVG so the drop shadow also rotates with it.
