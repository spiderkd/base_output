## The ghost text effect

The display number is rendered twice: the primary value at full opacity in `var(--riso-primary)`, and a ghost copy at 35% opacity in `var(--riso-secondary)` positioned `2px right, 2px down` using `position: absolute`. This creates the misregistration effect on the number itself.

The sparkline follows the same two-layer rule: a secondary ink polyline at 1.5px sits offset behind the primary 2px polyline. Both use `strokeLinecap="square"` — no rounded ends.
