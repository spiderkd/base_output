## Design notes

The toggle thumb is a **square** — no border-radius. It slides via `transform: translateX()` driven by a CSS transition. ON/OFF labels are SVG `<text>` elements that swap visibility with the thumb position. The track misreg shadow is a second absolutely-positioned div in secondary color, offset `3px` down-right.
