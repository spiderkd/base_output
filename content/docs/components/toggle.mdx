## Props

The examples above cover the most common ways to use `Toggle`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `label` | `string` | - | Optional label text. |
| `size` | `"sm" \| "md" \| "lg"` | `"md"` | Controls the overall component scale. |

`Toggle` also accepts the usual checkbox-style props such as `checked`, `defaultChecked`, `onChange`, `name`, and `disabled`.

## Design notes

The toggle thumb is a **square** — no border-radius. It slides via `transform: translateX()` driven by a CSS transition. ON/OFF labels are SVG `<text>` elements that swap visibility with the thumb position. The track misreg shadow is a second absolutely-positioned div in secondary color, offset `3px` down-right.

That restraint is what keeps `Toggle` feeling printed instead of purely digital: the offsets stay small, the edges stay sharp, and the ink contrast does most of the visual work.
