## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `items` | `BreadcrumbItem[]` | — | `{ label, href?, onClick? }[]` |
| `maxItems` | `number` | `4` | Max visible before collapsing to `···` |

## Design notes

The separator is two SVG `<line>` elements: a 2.5px primary diagonal slash (`10,2 → 4,18`) and a 1px secondary slash offset 2px right (`12,4 → 6,20`).

When collapsed, the ellipsis button renders three 4×4 squares in alternating primary, secondary, and overlap colors. Clicking expands the full path.

Active (last) item has a 2px primary underline. Parent items are secondary color with no underline.
