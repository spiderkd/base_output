## Props

The examples above cover the most common ways to use `Alert`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `variant` | `AlertVariant` | `"info"` | Visual variant for the component. |
| `title` | `string` | - | Heading shown at the top of the component. |
| `children` | `React.ReactNode` | - | Body content rendered inside the alert. |
| `dismissible` | `boolean` | `false` | Configures `dismissible` for the component. |
| `onDismiss` | `() => void` | - | Configures `onDismiss` for the component. |

This component also accepts the shared `theme`, `primary`, `secondary`, `overlap`, and `paper` overrides.

## Design notes

The 5px left accent bar is the core structural element — a solid strip in the accent color running the full height. The `danger` variant overlays a 45° diagonal cross-hatch `<pattern>` at 12% opacity over the red fill, simulating a warning print texture.

That restraint is what keeps `Alert` feeling printed instead of purely digital: the offsets stay small, the edges stay sharp, and the ink contrast does most of the visual work.
