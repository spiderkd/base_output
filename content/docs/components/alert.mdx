## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `"info" \| "success" \| "warning" \| "danger"` | `"info"` | Visual style |
| `title` | `string` | — | Bold headline |
| `dismissible` | `boolean` | `false` | Show close button |
| `onDismiss` | `() => void` | — | Called on dismiss |

## Design notes

The 5px left accent bar is the core structural element — a solid strip in the accent color running the full height. The `danger` variant overlays a 45° diagonal cross-hatch `<pattern>` at 12% opacity over the red fill, simulating a warning print texture.
