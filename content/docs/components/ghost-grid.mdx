## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `count` | `number` | `6` | How many section numbers to show |
| `opacity` | `number` | `0.05` | Base opacity of numerals |
| `stagger` | `boolean` | `true` | Alternate primary/secondary per numeral |
| `children` | `ReactNode` | — | Content rendered above the ghost layer |

## Design notes

Each numeral is rendered twice — a secondary ghost copy at `top: 3px, left: 3px` and the primary numeral at `position: relative`. Font size uses `clamp(80px, 12vw, 180px)` to scale with the viewport.

`stagger` alternates which ink color gets the primary treatment per numeral — even indices use primary + secondary ghost, odd indices reverse — creating a two-ink texture.

Everything is `userSelect: none` and `pointer-events: none`. The content layer sits at `z-index: 1`. Zero JavaScript. Server-renderable.
