## Design notes

The misregistration effect is achieved with an absolutely-positioned `div` in the secondary ink color, offset `2px right, 2px down` behind the input frame. On focus, the outline shifts from secondary → primary color, simulating the second ink plate snapping into sharper registration.

There is intentionally **no border-radius**. All riso components use `0px` corners — the only exception is pill badges.
