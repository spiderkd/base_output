## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `value` | `Date` | — | Controlled selected date |
| `defaultValue` | `Date` | — | Uncontrolled default |
| `onChange` | `(date: Date) => void` | — | Change handler |
| `events` | `CalendarEvent[]` | — | `{ date: "YYYY-MM-DD", label, ink? }[]` |

## Design notes

Selected date: solid `var(--riso-primary)` fill — flat ink square covering the cell. No border-radius.

Today (if not selected): 2px primary outline with `drop-shadow(2px 2px 0 var(--riso-secondary))`.

Event indicators are 5×5 squares (not dots) below the date number, with a negative `2px` bottom margin making them bleed past the cell boundary.

Month header lives in a PrintBlock (solid primary fill). Day-of-week headers are secondary uppercase labels. Weekends are secondary color numbers.
