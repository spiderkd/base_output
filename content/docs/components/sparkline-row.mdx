## Inside a table cell

```tsx
<Table
  columns={[
    { key: "label", header: "Metric" },
    {
      key: "trend",
      header: "7-day trend",
      render: (_, row) => (
        <SparklineRow data={row.trendData} width={80} height={24} />
      ),
    },
    { key: "value", header: "Latest", align: "right" },
  ]}
  data={metrics}
/>
```

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `number[]` | — | Array of values (min 2) |
| `width` | `number` | `80` | SVG width in px |
| `height` | `number` | `28` | SVG height in px |
| `ink` | `"primary" \| "secondary"` | `"primary"` | Line color |

## Design notes

A minimal SVG — shadow polyline (1px, 45% opacity, offset 1.5px) behind the primary line (2px). Both use `strokeLinecap="square"`. The final data point gets a 6×6 square marker — no circle, consistent with all riso chart data points.

Requires no wrapper div. Set `width` and `height` directly.
