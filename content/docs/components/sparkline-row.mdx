## Props

The examples above cover the most common ways to use `SparklineRow`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `data` | `number[]` | - | Series used for the miniature trend line. |
| `width` | `number` | `80` | Sparkline width in pixels. |
| `height` | `number` | `28` | Sparkline height in pixels. |
| `ink` | `"primary" \| "secondary"` | `"primary"` | Ink channel used for the line. |

This component also accepts the shared `theme`, `primary`, `secondary`, `overlap`, and `paper` overrides.

## Design notes

A minimal SVG — shadow polyline (1px, 45% opacity, offset 1.5px) behind the primary line (2px). Both use `strokeLinecap="square"`. The final data point gets a 6×6 square marker — no circle, consistent with all riso chart data points.

Requires no wrapper div. Set `width` and `height` directly.

That restraint is what keeps `SparklineRow` feeling printed instead of purely digital: the offsets stay small, the edges stay sharp, and the ink contrast does most of the visual work.
