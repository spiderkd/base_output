## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `number[][]` | — | 2D array of values 0–1 |
| `rowLabels` | `string[]` | — | Left-side row labels |
| `colLabels` | `string[]` | — | Top column labels |
| `cellSize` | `number` | `36` | Cell width and height in px |

## Design notes

This is the defining riso-unique data display. Each cell gets a unique SVG `<pattern>` with a dot radius mapped from the value (0.8px → 5px). At high values dots overlap, creating the overlap ink color naturally — the paper shows through less.

Hover shows the numeric percentage (0–100) centered in the cell and jumps the outline to 2px. Hover shadow is a secondary-color div offset `2px right, 2px down` at 35% opacity.
