## Props

The examples above cover the most common ways to use `Paper Texture Card`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `grain` | `"fine" \| "medium" \| "heavy" \| "newsprint"` | `"medium"` | Paper grain preset used in the background texture. |
| `tint` | `boolean` | `false` | Adds a light secondary-ink tint to the paper. |
| `halftone` | `boolean` | `false` | Adds visible halftone dots over the texture. |
| `children` | `React.ReactNode` | - | Content rendered inside the paper card. |

## The frequency difference

The key difference from the grain used in other components:

| Usage                          | `baseFrequency` | Visual                      |
| ------------------------------ | --------------- | --------------------------- |
| Button, Card borders           | `0.65`          | Fine grain — barely visible |
| Paper Texture Card (fine)      | `0.65`          | Same, explicit              |
| Paper Texture Card (medium)    | `0.40`          | Visible texture             |
| Paper Texture Card (heavy)     | `0.25`          | Coarse texture              |
| Paper Texture Card (newsprint) | `0.08`          | Visible fiber/grain         |

The `newsprint` level produces visible directional texture that resembles paper fiber — appropriate for editorial or zine-style layouts.

## Design notes

That restraint is what keeps `Paper Texture Card` feeling printed instead of purely digital: the offsets stay small, the edges stay sharp, and the ink contrast does most of the visual work.
