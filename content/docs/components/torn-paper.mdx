## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `seed` | `number` | `42` | PRNG seed — same seed = same tear |
| `height` | `number` | `48` | Total element height in px |
| `fill` | `"paper" \| "primary" \| "secondary" \| "overlap" \| "none"` | `"paper"` | Fill color above the tear |
| `flipY` | `boolean` | `false` | Flip vertically — tear hangs down from above |

## How it works

The tear path is generated by Mulberry32 — a 4-line deterministic PRNG seeded by `seed`. The same seed always produces the same path, so the component re-renders without the tear "jumping" to a new shape.

40 points along the x-axis each get a random y offset of ±35% of element height. These are connected with cubic bezier segments. Two paths are drawn: primary at `midY + noise`, secondary shadow at `midY + noise + 2px` — both from the same sequence so they run perfectly parallel.
