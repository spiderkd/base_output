## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `{ label: string, value: number }[]` | — | Chart data |
| `height` | `number` | `180` | SVG height in px |
| `showArea` | `boolean` | `true` | Halftone area fill |
| `showGrid` | `boolean` | `true` | Dotted horizontal grid |
| `showDots` | `boolean` | `true` | Square data point markers |

## Design notes

The area fill is a `<polygon>` filled with a `<pattern>` of 1px circles at 4px spacing — a halftone wash rather than a transparent alpha fill.

Data points are 6×6 `<rect>` squares, not circles. All riso chart data points are squares consistently.

On hover, a dashed vertical guide appears at the hovered x-position and the value floats above the data point in primary color.
