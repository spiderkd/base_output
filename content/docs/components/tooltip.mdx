## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `content` | `ReactNode` | — | Tooltip text or content |
| `placement` | `"top" \| "bottom" \| "left" \| "right"` | `"top"` | Position relative to trigger |
| `children` | `ReactElement` | — | The element that triggers the tooltip |

## Design notes

The tooltip body is a solid primary ink rectangle — no border-radius, no blur, no fade-in. It appears in 80ms and disappears in 80ms because riso ink doesn't fade.

The arrow pointer is two overlapping SVG `<polygon>` elements: a secondary shadow triangle offset `2px right, 2px down` behind the primary triangle. This gives the pointer the same misregistration depth as every other component.

The `2px` secondary shadow behind the main tooltip box is lighter than the button's `4px` shadow — tooltips sit closer to the surface than interactive elements.
