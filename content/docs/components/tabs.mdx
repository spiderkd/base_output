## Design notes

The active tab has `background: var(--riso-primary)` and sits 2px lower than the tab list border — visually merging with the content below it. Inactive tabs are transparent with primary-colored text.

The secondary misregistration line runs the entire width of the tab list at `translateY(2px)` below the primary border — visible at the ends of the list where no tab covers it. This detail is the risograph effect: the second ink plate never quite lines up.

There is intentionally **no border-radius** on any tab. All corners are sharp 90°.
