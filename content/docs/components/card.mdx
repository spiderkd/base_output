## Props

The examples above cover the most common ways to use `Card`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `padding` | `number` | `20` | Configures `padding` for the component. |
| `stacked` | `boolean` | `false` | Adds the printed-sheet offset stack effect. |
| `filled` | `boolean` | `false` | Uses a solid ink fill with inverse text. |
| `halftone` | `boolean` | `false` | Adds a halftone texture inside the card. |
| `interactive` | `boolean` | `false` | Enables hover and press styling. |
