## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `data` | `ChartSegment[]` | — | Array of `{ label, value, ink? }` |
| `size` | `number` | `200` | Diameter in px |
| `donut` | `boolean` | `true` | Show center hole |
| `showLegend` | `boolean` | `true` | Show legend below chart |

## Design notes

Each segment is an SVG arc path with `style={{ mixBlendMode: "multiply" }}`. Where two segments share a boundary, their colors multiply — primary pink × secondary teal = overlap purple, automatically, from the blend mode. No explicit overlap color is set.

The `feTurbulence` grain filter is applied per segment giving each a slightly different texture. Hover lifts a segment with `drop-shadow(4px 4px 0 var(--riso-overlap))`.
