## Props

The examples above cover the most common ways to use `Separator`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `variant` | `SeparatorVariant` | `"default"` | Visual variant for the component. |
| `orientation` | `SeparatorOrientation` | `"horizontal"` | Configures `orientation` for the component. |
