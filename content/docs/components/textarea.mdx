## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `label` | `string` | — | Field label |
| `hint` | `string` | — | Helper text below |
| `error` | `string` | — | Error text (turns label red) |
| `showCount` | `boolean` | `false` | Show character count |
| `maxLength` | `number` | — | Character limit |

## Design notes

Extends the `Input` misregistration pattern. On focus, the secondary shadow opacity transitions from `0.22` to `0.35` and the outline switches from secondary to primary color.

The browser's native resize handle is replaced by two SVG `<line>` elements: one primary (1.5px) and one secondary (1px, offset 1.5px). The custom handle preserves resize functionality while staying on-aesthetic.
