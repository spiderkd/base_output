## Props

The examples above cover the most common ways to use `Checkbox`. Start with the default example, then layer in the props below when you need more control.

| Prop | Type | Default | Description |
| -------------- | ------------------------- | ----------- | ------------------------------- |
| `label` | `string` | - | Optional label text. |
| `description` | `string` | - | Optional supporting text. |

`Checkbox` also accepts the usual checkbox input props such as `checked`, `defaultChecked`, `onChange`, `name`, and `disabled`.
