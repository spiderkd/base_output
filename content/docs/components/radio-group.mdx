## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `options` | `RadioOption[]` | — | `{ value, label, description?, disabled? }[]` |
| `value` | `string` | — | Controlled value |
| `defaultValue` | `string` | — | Uncontrolled default |
| `onChange` | `(value: string) => void` | — | Change handler |
| `label` | `string` | — | Group label |
| `orientation` | `"vertical" \| "horizontal"` | `"vertical"` | Layout direction |

## Design notes

The radio indicator is two concentric SVG circles — the same geometry as the Avatar component. Outer circle: 2px primary border. Inner: paper background idle, floods with primary on select. A secondary shadow circle (1.5px, offset 2.5px) sits behind the outer ring.

RadioGroup uses circles rather than the squares used everywhere else in the system. Circles are reserved for radio buttons and avatars — the only rounded shapes in Riso UI (aside from pills and stamps).
