## Node states

| Status | Visual |
|---|---|
| `complete` (`idx < currentStep`) | Solid primary fill, white checkmark, secondary shadow circle |
| `current` (`idx === currentStep`) | Primary outline, secondary shadow, inner filled dot |
| `upcoming` (`idx > currentStep`) | Dashed `strokeDasharray="3 2.5"` primary circle at 50% opacity |

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `steps` | `Step[]` | — | `{ id, label, description? }[]` |
| `currentStep` | `number` | — | 0-based index of the current step |
| `orientation` | `"horizontal" \| "vertical"` | `"horizontal"` | Layout direction |

## Design notes

Connector lines between steps use the double-rule pattern: 2px primary + 1px secondary offset 3px. Completed connectors are solid at full opacity. Upcoming connectors drop to 35% opacity. No transition on the connector — it changes immediately when `currentStep` updates, consistent with the stamp's instantaneous quality.
