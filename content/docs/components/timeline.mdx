## Item status reference

| Status | Node | Content card |
|---|---|---|
| `complete` | Solid primary + white checkmark | Light tint outline |
| `current` | Primary outline + inner dot | Full `drop-shadow(3px 3px 0 var(--riso-secondary))` |
| `upcoming` | Dashed primary outline | Reduced opacity outline |

## Design notes

The spine is two vertical lines: 2px primary + 1px secondary offset 4px right at 70% opacity — the double-rule applied vertically.

Date labels sit left of the spine in secondary uppercase. Content cards sit right. `current` cards get the full hard shadow treatment. Dates use `white-space: nowrap` and `text-align: right`.
