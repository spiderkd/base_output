## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `variant` | `"target" \| "corner" \| "cross" \| "full-bleed"` | `"target"` | Mark type |
| `corner` | `"tl" \| "tr" \| "bl" \| "br"` | `"tl"` | Used with `variant="corner"` |
| `size` | `number` | `48` | Width/height in px |
| `primaryOpacity` | `number` | `0.8` | Primary ink opacity |
| `secondaryOpacity` | `number` | `0.55` | Secondary ink opacity |

## Design notes

In real riso printing, registration marks are printed in both ink passes so the operator can check alignment. The offset between the two renderings shows the misregistration. This component decorates your UI with those same marks — a nod to the physical print process.

All marks use two-layer SVG: primary ink (full weight) + secondary ink (lighter, offset 2px right/down). The `full-bleed` variant uses `position: absolute; inset: 0` and places a `CornerMark` at each corner.
